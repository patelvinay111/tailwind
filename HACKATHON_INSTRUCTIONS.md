## 2. Vocal Bridge (Voice AI Layer)

**Promo Code:** `VBHACKMONTH` (free month of Developer Plan)

1. Sign up or log in: vocalbridgeai.com/auth/signup
2. Open dashboard: vocalbridgeai.com/app/dashboard
3. Open billing, choose Developer Plan: vocalbridgeai.com/app/billing
4. At Stripe checkout, use promo code `VBHACKMONTH`
5. **Set a calendar reminder** — subscription renews automatically next month

### Create Agent via UI
1. Click Create Agent: vocalbridgeai.com/app/agents/new
2. Choose: Web only, Phone only, or Both
3. Fill in: name, style, greeting, system prompt
   - Optional: AI agent integration mode (if you have an existing AI agent)
   - Optional: Outbound calling (if your app needs to call users)
4. Click Create Agent
5. Open Test tab and talk to it
6. For app integration: open Develop, create/copy API key
7. Developer Guide: vocalbridgeai.com/app/developer-guide
8. Share with teammates from agent menu

### Create Agent via Coding Assistant
1. Go to API keys, create new key
2. Copy full Developer Guide: vocalbridgeai.com/app/developer-guide
3. Paste into Cursor / Claude Code / coding assistant
4. Use prompt to build voice integration
5. Run app and test voice flow
